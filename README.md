# playwright-pom-automate

## Getting Started

### Prerequisites

- Node.js installed on your local machine
  
### Installation

1. **Clone the Repository**: 

   ```bash
    https://github.com/elzeslymai-1/playwright-pom-automate.git
   ```
   
2. **Navigate to the Project Directory**:

   ```bash
   cd playwright-pom-automate
   ```
4. **Install Dependencies**:
   
   ```bash
   npm install
   ```
  
     
### Usage

1. **Add .env file following .env.example**:
   

2. **Run test on Android**:
   
   ```bash
   npm run test:ui
   ```

2. **Show Report**:
   
   ```bash
   npm run report
   ```
